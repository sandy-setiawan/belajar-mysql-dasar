# Belajar MySQL Bahasa Indonesia

## Pengenalan MySQL

MySQL merupakan sistem basis data (Database Management
System) atau aplikasi yang digunakan untuk membuat, 
menyimpan, dan mengelola data secara efisien.

Jika menggunakan analogi, MySQL dapat dianggap sebagai
aplikasi semacam Microsoft Excel, di mana ketika kita
membuat file Excel akan sama seperti membuat database
di MySQL. Di dalam file tersebut, kita bisa membuat
banyak Sheet, di mana hal ini bisa dianalogikan sebagai
tabel pada aplikasi MySQL. Tabel pada MySQL juga konsepnya
mirip seperti tabel di aplikasi Microsoft Excel, terdiri
dari baris dan kolom.
