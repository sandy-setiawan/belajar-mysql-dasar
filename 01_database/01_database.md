# 1. Membuat Database di MySQL

Pada folder ini, akan disajikan cara untuk membuat database
pertama di aplikasi MySQL. Pastikan untuk menyalakan aplikasi
MySQL Anda, baik menggunakan XAMPP atau Laragon.

Setelah Anda masuk, database dapat dibuat menggunakan perintah:

```CREATE DATABASE nama_database;```

Setelah database dibuat, database tersebut tidak serta merta
langsung dapat digunakan. Anda harus terlebih dahulu masuk
ke dalamnya, dengan cara menjalankan perintah:

```USE nama_database;```

Selamat! Anda baru saja membuat database pertama Anda.
Untuk menghapus database, Anda dapat menjalankan perintah:

```DROP DATABASE nama_database;```
